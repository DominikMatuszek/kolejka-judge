\subsection{Limits and privilege deescalation}

\begin{minted}[linenos]{python}
checking.add_steps(
    solution=RunJavaClassSolution(
        class_name='Solution',
        stdout='out',
        limits={
            'time': 2.5,
            'memory': 1024 * 1024 * 10,
        },
        user='nobody',
    ),
)
\end{minted}

The \code{RunJavaClassSolution} class declares \hyperref[subsec:UsedTimePostcondition]{\python{UsedTimePostcondition}}
and \hyperref[subsec:UsedMemoryPostcondition]{\python{UsedMemoryPostcondition}}, therefore if the real running time will
exceed 2.5 seconds, \code{TLE} exit status will be returned.
Similarly, if the total RAM usage exceeds 10 MB, checking will return \code{MEM}.
If the script is running as a superuser, dropping privileges is useful -- in this case, user called \code{nobody} will
be assigned as the process owner via the \code{setuid} call.
